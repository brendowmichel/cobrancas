# Cobrancas

Sistema de gestao de cobrancas em Next.js + Supabase.

## Desenvolvimento

1. Instale as dependencias:

```bash
npm install
```

2. Crie `.env.local` baseado em `.env.example`:

```bash
SUPABASE_URL=https://seu-projeto.supabase.co
SUPABASE_SERVICE_ROLE_KEY=sua_service_role_key
```

3. Rode localmente:

```bash
npm run dev
```

## Deploy na Vercel

Cadastre as mesmas variaveis em:

```text
Project Settings > Environment Variables
```

O projeto legado em Google Apps Script foi movido para:

```text
backup/gas-legacy-2026-06-24
```
